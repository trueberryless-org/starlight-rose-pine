The Starlight Rosé Pine theme can be configured inside the `astro.config.mjs` configuration file of your project:

```js {11}
// astro.config.mjs
import starlight from "@astrojs/starlight";
import { defineConfig } from "astro/config";
import starlightRosePine from "starlight-rose-pine";

export default defineConfig({
  integrations: [
    starlight({
      plugins: [
        starlightRosePine({
          // Configuration options go here.
        }),
      ],
      title: "My Docs",
    }),
  ],
});
```

## Configuration options

The Starlight Rosé Pine theme accepts the following configuration options:

### `dark`

**Type:** `{ flavor?: string; accent?: string }`<br />
**Default:** `{ flavor: "main", accent: "pine" }`

Configure the theme for the dark mode.

#### `flavor`

**Type:** `string`<br />
**Default:** `main`

Configure the flavor for the dark theme.
Available values are:

- `"main"`
- `"moon"`

#### `accent`

**Type:** `string`<br />
**Default:** `pine`

Configure the accent for the dark theme.
Available values are:

- `"love"`
- `"gold"`
- `"rose"`
- `"pine"`
- `"foam"`
- `"iris"`

### `light`

**Type:** `{ flavor?: string; accent?: string }`<br />
**Default:** `{ flavor: "dawn", accent: "pine" }`

Configure the theme for the light mode.

#### `flavor`

**Type:** `string`<br />
**Default:** `dawn`

Configure the flavor for the light theme.
Available values are:

- `"dawn"`

#### `accent`

**Type:** `string`<br />
**Default:** `pine`

Configure the accent for the light theme.
Available values are:

- `"love"`
- `"gold"`
- `"rose"`
- `"pine"`
- `"foam"`
- `"iris"`
